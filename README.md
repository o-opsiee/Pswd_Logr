# Pswd_Logr

~ What's this?
  This is a script to log or save or write passwords in a txt file.

~ Why did i build this?
~ I built this to automate password saving on my local storage, particularly for my bandit labs from overthewire. Earlier, i had to create/open the txt file and type the level and corresponding password manually for each level. Now, with this itty bitty program i just have to "python3 pswd_logr.py" my CLI. It asks for level and password then saves it in the file.

~ Programming language used?
~ Python.
